Add a Table Component

  1. In the Component Palette, drag a Table component and drop it on the page.
  2. Use the Table component's Add Data Quick Start to add data to your table.

Add a Rating Gauge Component

  1. In the Component Palette, drag a Rating Gauge component and drop it on the Rating column of the Table component.
  2. In the Data tab of the Properties pane for the Rating Gauge component, set the following property values:
    • Value: [[$current.data]]
    • selected-state.color: "[[ $functions.getRatingColor($current.data) ]]"

    This adjusts the rating gauge color based on the value in the Rating column.

  3. In the JavaScript tab, add a new function getRatingColor( value ) and return a custom color based on the input value. 
    PageModule.prototype.getRatingColor = function (value) {
    if (value >= 0 && value <= 1) {
      return 'red';
    } else if (value > 1 && value <= 2) {
      return 'pink';
    } else if (value > 2 && value <= 3) {
      return 'orange';
    } else if (value > 3 && value <= 4) {
      return 'blue';
    } else if (value > 4) {
      return 'green';
    }
  }
