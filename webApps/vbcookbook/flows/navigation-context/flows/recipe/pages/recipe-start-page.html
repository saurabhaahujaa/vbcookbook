Create Employee Table
  1. In the Component Palette, drag and drop a Table component onto the page.
  2. Use the Table component's Add Data Quick Start to configure data for the Table component:
    1. Select the Employee business object as the endpoint.
    2. Select the following fields from the Response:
      • id
      • firstName
      • lastName
      • email
    3. Click Next and then click Finish.
Add a Navigation Toolbar
  1. In the Variables tab of the Page Designer, create a new variable of type Object, id of selection, and add the following fields:
    • key of type Number
    • data of type get_Employee. You can create get_Employee from the Employee business object's get_Employee endpoint.
  2. In the Component Palette, drag and drop a Toolbar component onto the page.
  3. Drag and drop a Button component into the Toolbar component and set the value of Text to Navigate to Edit With Employee ID.
  4. Drag and drop a Button component into the Toolbar and set the value of Text to Navigate To Detail with Employee.
  5. In the Properties pane, set both Buttons' Disabled property to [[!$variables.selection.key]].
  6. In the Properties pane of the Table component, set the Selection Mode, Row property to Single.
  7. In the Data tab, set the value of the First Selected Row property to {{ $page.variables.selection }}.
 Create Edit and Detail Pages
  1. Use the Table component's Add Edit Quick Start to create an edit page.
  2. Create a new id variable of type Number in the new edit page, set the Input Parameter to Required, and select the Pass on URL checkbox.
  3. Use the Table component's Add Detail Quick Start to create a new detail page.
  4. Create a new employee variable of type get_Employee in the new detail page, and set the Input Parameter to Required.
 Navigate to Pages
  1. In the start page, add an ojAction event listener to the Navigate to Edit With Employee ID button.
  2. In the action chain, drag and drop a Navigate action.
  3. In the Properties pane of the Navigate action, set the target to the edit page.
  4. Map the input id to $page.variables.selection.key.
  5. Add an ojAction event listener to the Navigate To Detail with Employee button.
  6. In the action chain, drag and drop a Navigate action.
  7. In the Properties pane of the Navigate action, set the target to the detail page.
  8. Map the input employee to $page.variables.selection.data.
