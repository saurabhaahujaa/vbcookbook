Add UI Components to Page
  1. From the Component Palette, drag a Button component and drop it on the page.
  2. From the Component Palette, drag a Text Area component and drop it on the page.

    This component displays the user's location when it is retrieved.

 Create Action Chain on Button
  1. Add an ojAction event listener to the Button component.
  2. In the action chain, add a Get Location action right after the Start node.
 Assign Results to Variables
  1. In the Variables tab of the Page Designer, create two variables of type string:
    • latitude
    • longitude
  2. In the action chain, drop an Assign Variables action under success.
  3. Click Assign and on the left side expand Results to find:
    • Accuracy
    • Altitude
    • Altitude Accuracy
    • Heading
    • Latitude
    • Longitude
    • Speed
  4. Map appropriate results to the variables you created, and click Save.
 Display Results
  1. In the Page Designer, select the Text Area component.
  2. In the Data tab of the Properties pane, use the fx to add the following values to Text Area {{"User Current Location is "+$variables.latitude + ", "+ $variables.longitude }}.
