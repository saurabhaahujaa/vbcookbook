This recipe shows you how to create a form control within a Validation Group component (oj-validation-group) where the action chain triggered by the component event calls a JavaScript function that uses Oracle JET's AsyncValidator API. The recipe does not connect to a backend REST service to asynchronously validate user input, but instead simulates the connection, and creates a number range validator with minimum and maximum values for the user input.
