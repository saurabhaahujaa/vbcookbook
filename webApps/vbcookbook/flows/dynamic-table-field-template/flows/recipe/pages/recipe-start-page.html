To use this recipe, you need to install components from the Component Exchange. Your Visual Builder administrator must have configured your Visual Builder instance for you to be able to work with the Component Exchange.

Create a Dynamic Table

  1. Install the JET Dynamic Table component from the Component Exchange, then drag and drop it on the template in the Page Designer.
  2. In the Quick Start tab of the Properties pane, click Configure Dynamic UI.
  3. In the Configure Dynamic UI dialog that appears, choose the business object or REST endpoint that you want to use as the data source for the form, and click Next.
  4. In the Select Rule Set page of the Configure Dynamic UI dialog, click New Rule Set to define your own display logic, and enter a rule set label and ID, such as Simple Employee List RuleSet.
  5. In the Fields section, select the following fields, and click Next, and then click Finish.
    • firstName
    • lastName
    • phoneNumber
    • email
    • salary

Create Field Templates for the Dynamic Table

  1. In the Dynamic UIs tab, select Employee.
  2. Open the Simple Employee List RuleSet that you created when creating the Dynamic Table.
  3. In the Fields tab, click + Custom Field to create a new custom field with an ID value of fullName, type String, and a Value property that references [[$fields.firstName.value() + ' ' + $fields.lastName.value()]] .
  4. In the Rule Sets tab for the Simple Employee List RuleSet, open the default layout and select the email field in the list of fields.
  5. In the Properties pane, click Create beside the Template field to create a new template with a value of Email Template for the Label property and emailTemplate for the ID property.
  6. Open the Code view of the template page, drag an Icon component and drop it above the oj-input-text component and, in the General tab of the Properties pane, click the Image icon to open the Icon Gallery where you can select an envelope image.
  7. Repeat these steps to also add a template for the phoneNumber field.
