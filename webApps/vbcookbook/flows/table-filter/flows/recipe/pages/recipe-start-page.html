A Table component that displays department data. An Input Text component assigns the value that the user enters to a page variable, and the page variable is used by the {filterCriterion}.

Create the Page Model
  1. Create a page variable of type String named filterText.
 Create the UI

Add an Input Text component for the user to enter the text value that will filter the Table componentâ€™s data.

  1. Add the Input Text and Label components to the page.
  2. In the Data Tab of the Properties pane, bind the Value property to $variables.filterText.
  3. Add the Table component below the Input Text and Label.
  4. Optionally use cellTemplate and oj-highlight-text in table to highlight the matching text.
 Bind the Table Component to Data
  1. With the Table component selected, use the Add Data Quick Start to bind the Table component to the GET /Departments endpoint.
  2. In the Bind Data page of the Add Data dialog, add the following fields to the Table component:
    • id
    • departmentName
    • departmentManager.lastName
    • departmentManager.email
  3. In the Define Query page, click the {filterCriterion} variable under Target and add the following conditions:
    • id $co $variables.filterText
    • departmentName $co $variables.filterText
    • departmentManager.lastName $co $variables.filterText
    • departmentManager.email $co $variables.filterText
  4. Select Match Any.
