When dealing with large quantity of data, it is often a good idea to load data as per demand via pagination. Out of the box, List view and Tables can work with Service Data Provider to load chunks of data as required.

Create a List View With Employee BO
  1. Drag and Drop a List View component from the component palette onto the design view
  2. Use Quick Start to add Data
 Configure Pagination on List
  1. Use the property inspector to locate the scroll policies
  2. Set the scroll-policy to loadMoreOnScroll
  3. Expand the scroll-policy-options to set values for the below
    • fetch-size to 15
    • max-count to 1000
    • For loadMoreOnScroll to work properly, the scroller must be visible, set max-height to a fixed value via style property.
