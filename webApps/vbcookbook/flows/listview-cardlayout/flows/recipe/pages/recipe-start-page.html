A list view displays employees records in card layout format.

Create Card Layout

You add a List View component to your page, bind it to employee data and add some styling to render data in nice looking cards format

  1. In the Component Palette, drag a List View component and drop it on the page.
  2. Use the List View componentâ€™s Add Data Quick Start to bind your list to employee data.
  3. In the Bind Data page, select the following employee fields:
    • First Name
    • Last Name
    • Email
    • Salary
  4. Add some style class at resources/css/app.css.
    Note: This demo showcase a sample look and feel to display cards. Your usecase might require different user experience, so you can add your own styles and html elements to create your own demo
