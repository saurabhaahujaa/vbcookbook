Summary

  • Create Business Objects.
  • Create a ListView component.
  • Bind the Department BO data into the ListView component. Select partial fields of the BO and related BOs.
Step 1: Create Business Object
  1. Create Department and Location Business Object
 Step 2: Add ListView UI component to page and populate data
  1. Create a new page and drag and drop a ListView component from Component Palette.
  2. From Quick Start menu select Add Data for the ListView component.
  3. From the list of data sources select Department BO.
  4. When configuring the fields to be displayed in the ListView select departmentName, and locationObject.city. Finish the wizard.
  5. Run the page. The ListView gets populated with the fields selected.
  6. Observe in the chrome console->network tab that the REST call made to fetch the data makes use of 'fields' query parameter to fetch only the required fields. The required fields are determined by the type definition of 'getallDepartmentResponse'. Should you need to modify the data required to populate the UI, you must update the definition of this data type.
