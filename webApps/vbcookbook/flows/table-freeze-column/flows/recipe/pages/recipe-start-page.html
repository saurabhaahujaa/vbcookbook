You add a Table component to your application that displays employee data and you configure the component to freeze columns at the start and the end of the table.

Create an Employee Table
  1. In the Component palette, drag a Table component and drop it on the page.
  2. In the Quick Start tab of the Properties pane, use the Table componentâ€™s Add Data quick start to bind your table to employee data.
  3. In the Bind Data page, select the following employee fields:
    • First Name
    • Last Name
    • Email
    • Phone Number
    • Salary
    • Hire date
    • Job title
    • Manager
    • Created date
    • Created by
  4. Click Next and Finish to close the dialog.
 Configure the frozenEdge Property for Columns in the Table

To freeze a column in a Table component, you configure the Table Columns frozenEdge property in the Code view of the Designer with the appropriate value (start or end). For example, the following code snippet shows how you configure the First Name and Email columns to freeze at the start of the table and the Created By column to freeze at the end.

columns='[{"headerText":"First Name","field":"firstName", "frozenEdge": "start"},...,
          {"headerText":"Email","field":"email","frozenEdge": "start"},...,
          {"headerText":"Created By","field":"createdBy", "frozenEdge": "end"}]'>
