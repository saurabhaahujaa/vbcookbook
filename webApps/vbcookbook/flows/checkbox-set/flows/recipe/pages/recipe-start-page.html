Populate the Checkbox Set component with data from an SDP and use an array variable to track user selection.

Create the UI
  1. In the Component Palette, drag and drop a Checkbox Set component onto the page.
  2. Use the Checkbox Set component's Add Options Quick Start to select the endpoint structure for the Region business object where you select the id and regionName items as the values for the component's Value and Label properties.
  3. In the Variables tab, create a variable named selectedValues of type Number and set the value attribute of the component to selectedValues. The type of selectedValues is set to Number to match the type of the id attribute(s) for the SDP data rows.
  4. In the Component Palette, drag and drop two Button components above the Checkbox Set component on the page. Set the first Button component's Text property to Reset All and the second component's to Select Asia.
  5. Drag and drop a Text Area component under the Checkbox Set component and set its value attribute to reference the selectedValues variable.
Preselect a Checkbox Set Component Option During Page Load
  1. In the Variables tab, select the selectedValues variable and, in the Properties pane, set Default Value to [1] to select the first option in the Checkbox Set component when the page loads.
Reset Checkbox Set Component Selection
  1. Select the Reset All Button component and, in the Events tab of the Properties pane, add an ojAction event to invoke an action chain.
  2. Set the ID of the newly-created action to resetAllActionChain and add an Assign Variables action to select the selectedValuesvariable and set its value to []. In the Reset Target dropdown list for the variable, select empty.
Select an Option in the Checkbox Set Component
  1. Select the Select Asia Button component and, in the Events tab of the Properties pane, add an ojAction event to invoke an action chain.
  2. Set the ID of the newly-created action to selectAsiaActionChain and add an Assign Variables action to select the selectedValuesvariable and set its value to [3]. In the Reset Target dropdown list for the variable, select empty. The value 3 is the id value of Asia.
