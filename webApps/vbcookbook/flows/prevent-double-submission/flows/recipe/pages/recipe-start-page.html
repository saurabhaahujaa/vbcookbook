You'll add a Button component to your application's page and configure an action chain that disables the button and displays a different label ("Processing...") until an action completes.

Create a Form With a Button
  1. In the Component Palette, drag and drop an Input Text component and a Button component onto the page.
  2. In the Variables tab of the Page Designer, create the following variables:
    • buttonText of type String.
    • disableButton of type Boolean.
  3. In the Component Palette, drag and drop a Text component (oj-bind-text) onto the Button component and set the Value property of the Text component to [[ $variables.buttonText ]].
  4. Wrap the Text component with a span tag.
 Add an Event Listener and an Action Chain to Prevent Double Click
  1. In the Events tab of the Properties pane for the Button component, click New Event to create an ojAction event listener.
  2. In the action chain editor, add an Assign Variables action to map to the two page variables that you created previously.
  3. In the Assign Variables dialog, assign Processing.... to the buttonText variable in Target and set the disableButton variable in Target to true.
  4. Add a Call Function action to call a JavaScript function, addCountry, that waits 1500 milliseconds.
  5. Add another Assign Variables action to reset the buttonText and disableButton variables to their original values after the Call Function action completes.
